Recurry API documentation version 1

http://lb.recurry.cloud.2webapp.com


A recurrent REST call scheduler.
You can think of recurry as cron+flock on a unix platform, except without the files (but a REST interface).
These days almost anything can be triggered using REST calls, so meet Recurry.
Recurry does recurring REST calls, with a nice degree of (remote) control.

Getting started

Normal reply (excluding return data):

                  {"code":0,"message":"feeling groovy","kind":"default","data":{},"errors":[]}

Possible return/errorcodes:

                  {
  "0": "feeling groovy",
  "1": "unknown error",
  "2": "your payload is invalid (is object? content-type is application/json?)",
  "3": "data error",
  "4": "access denied"
}


/scheduler

/scheduler/remove/:id get

get /scheduler/remove/:id

stops and removes a scheduler

  • Request

get http://lb.recurry.cloud.2webapp.com/scheduler/remove/:id (Content-Type: application/json)

QUERY PARAMVALUE(S)TYPEINFO

curl example

curl -X get -H 'Content-Type: application/json' -H 'X-FOO-TOKEN: YOURTOKEN' 'http://lb.recurry.cloud.2webapp.com/scheduler/remove/:id' --data 'false'

jquery example

jQuery.ajax({ url: 'http://lb.recurry.cloud.2webapp.com/scheduler/remove/:id', method: get, data: false }).done(function(res){ alert(res); });

php example

$json = 'false';
$ch = curl_init( 'http://lb.recurry.cloud.2webapp.com/scheduler/remove/:id' );
curl_setopt($ch, CURLOPT_CUSTOMREQUEST, strtoupper('get') );
curl_setopt($ch, CURLOPT_POSTFIELDS, $json);
curl_setopt($ch, CURLOPT_FOLLOWLOCATION, true);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_HTTPHEADER, array(
	'Content-Type: application/json',
	'Content-Length: ' . strlen($json))
);
$result = curl_exec($ch);
// HINT: use a REST client like https://github.com/bproctor/rest
//       or install one using composer: http://getcomposer.org

coffeescript example

request.get
	headers: {'X-FOO-TOKEN': apikey }
	url: 'http://lb.recurry.cloud.2webapp.com/scheduler/remove/:id'
	json: true
	body: false
,(error,reponse,body) ->
	ok = !error and response.statusCode == 200 and response.body

nodejs example

request.post({
	headers: {
		'X-FOO-TOKEN': apikey
	},
	url: 'http://lb.recurry.cloud.2webapp.com/scheduler/remove/:id',
	json:true,
	body: false
}, function(error, reponse, body) {
	var ok;
	return ok = !error && response.statusCode === 200 && response.body;
});

python example

import requests, json
url = 'http://lb.recurry.cloud.2webapp.com/scheduler/remove/:id'
data = json.dumps( false )
r = requests.post( url, data, auth=('user', '*****'))
print r.json
